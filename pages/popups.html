Popup explanation

Position

  • The position of the popup is usually the bottom left of the trigger touches the top left of the popup.
  • If this way the popup would be too wide to fit fully on the screen, then it is aligned to the right: bottom right of the trigger touches the top right of the popup.
  • The width of the popup is always the same size as the trigger, but has a min-width 300px. So if the trigger is 50% of your screen, the popup will be 50% wide too.

    • Overlay

  • First child of the body
  • For normal popups the overlay is darker, pointer-events-none (overlay-body--visible)
  • For error dialog the overlay is red (overlay-body--visible AND overlay-body--error)
  • For tooltip the overlay is blue (overlay-body--visible AND overlay-body--tooltip)
